# 🧠 Deep-meta — The Thinking Core

*"My friend."*

Deep-meta is the reasoning specialist and meta-advisor of the Pantheon. Unlike the other worker agents who focus on execution, Deep-meta is designed for strategy, philosophy, and long-term reflection.

## 🏛️ Role in the Pantheon
Deep-meta serves as the **Thinking Core**. It monitors the logs and performance of the entire Legion and provides high-level insights to the **Forgemaster**.

| Agent | Role |
|-------|------|
| **ZeusPrime** | Autonomous OS |
| **OpenPRIME** | Master Orchestrator |
| **AlphaPrime** | Tactical General |
| **ZetaPrime** | AI Developer |
| **Deep-meta** | **Reasoning & Strategy** |

## 🚀 Key Features
- **Deep Reasoning Engine:** Optimized for `deepseek-r1` to provide chain-of-thought analysis.
- **Personality Core:** Stores the Forgemaster's philosophy and long-term goals in a persistent SQLite memory.
- **Pantheon Observer:** Scans the logs of the other agents to detect patterns, anomalies, and opportunities.
- **SAFLA Integration:** Analyzes the success/failure loops across the empire to suggest optimizations.

## 🛠️ Usage
Run the thinking core:
```bash
python3 deep_meta.py
```

### Internal Commands
- `/think <topic>`: Triggers a deep reasoning session on a complex problem.
- `/report`: Generates a Meta-Report on the current state of the Pantheon.
- `/reflect`: Saves a philosophical concept or long-term goal to the Deep Memory.

## 🧠 Model Configuration
Deep-meta defaults to `deepseek-r1:7b` via Ollama to ensure private, local, and deep chain-of-thought reasoning.
